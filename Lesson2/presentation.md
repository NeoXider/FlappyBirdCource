# Слайд 3: Список труб и Таймер
Сначала подготовим место для труб. В `__init__` добавляем пустой список и **таймер** — встроенный инструмент SpritePro:

```python
import random

# ... после создания птички и физики ...

# 3. Список труб + таймер спавна каждые 2 секунды
self.pipes = []
self.spawn_timer = s.Timer(2.0, self.spawn_pipes, repeat=True, scene=self)
self.spawn_pipes()  # Первая пара сразу

# 4. Флаг окончания игры
self.is_game_over = False
```

`s.Timer` — это как будильник. Он каждые 2 секунды вызывает функцию `spawn_pipes`. А `repeat=True` говорит ему звонить бесконечно!

# Слайд 4: Функция создания труб
Пишем функцию `spawn_pipes` внутри `GameScene`. Она создаёт **пару труб** с дыркой в случайном месте:

```python
def spawn_pipes(self):
    """Создаёт пару труб с дыркой в случайном месте."""
    gap_y = random.randint(200, 400)  # Центр дырки
    gap_size = 150
    pipe_x = 450  # За правым краем экрана
    
    # Верхняя труба (перевёрнутая на 180°)
    top = s.Sprite("pipe.png", pos=(pipe_x, gap_y - gap_size/2 - 300), size=(80, 600), scene=self, sorting_order=5)
    top.angle = 180
    # Нижняя труба
    bottom = s.Sprite("pipe.png", pos=(pipe_x, gap_y + gap_size/2 + 300), size=(80, 600), scene=self, sorting_order=5)
    
    self.pipes.extend([top, bottom])
```

**Что здесь происходит?**
- `random.randint(200, 400)` — каждый раз дырка на разной высоте!
- `pipe_x = 450` — труба создаётся за правым краем экрана (ширина экрана = 400), чтобы плавно заехать.
- `top.angle = 180` — переворачиваем верхнюю трубу вверх ногами.

# Слайд 5: Движение труб
Теперь в методе `update` напишем цикл, который двигает **все трубы** влево:

```python
# --- Двигаем все трубы влево и проверяем столкновения ---
for pipe in self.pipes:
    pipe.x -= 200 * s.dt
```

`s.dt` (delta time) — это время одного кадра. Умножая на него, мы получаем **одинаковую скорость** на быстрых и медленных компьютерах.

# Слайд 7: Очистка памяти (kill)
⚠️ **Проблема:** Трубы улетают за экран... и летят дальше. Через час у нас будет 1000 спрайтов за кадром, и компьютер зависнет!

**Решение:** Удаляем трубы, которые улетели далеко:
```python
# Удаляем трубы, улетевшие за экран
for pipe in self.pipes[:]:
    if pipe.x < -100:
        pipe.kill()
        self.pipes.remove(pipe)
```
*Обратите внимание:* `self.pipes[:]` — это копия списка. Без неё нельзя удалять элементы прямо во время перебора!

# Слайд 8: Столкновения!
Как понять, что птичка ударилась? В SpritePro есть метод `.collides_with()` — он проверяет, пересеклись ли текстуры двух спрайтов.

Добавляем в тот же цикл движения:
```python
for pipe in self.pipes:
    pipe.x -= 200 * s.dt
    if self.player.collides_with(pipe):
        self.trigger_game_over()
```

И проверяем выход за границы экрана:
```python
# Смерть от выхода за границы экрана
if self.player.y > s.WH[1] or self.player.y < 0:
    self.trigger_game_over()
```

# Слайд 9: Game Over! (TextSprite)
Пишем функцию `trigger_game_over`. Когда птичка врезается, мы:
1. Останавливаем физику птички.
2. Показываем красную надпись `ИГРА ОКОНЧЕНА`.

```python
def trigger_game_over(self):
    """Останавливает игру и показывает надпись."""
    if self.is_game_over:
        return
    self.is_game_over = True
    self.player_body.velocity.y = 0
    s.TextSprite("ИГРА ОКОНЧЕНА", pos=s.WH_C, font_size=40, color=(255, 0, 0), scene=self, sorting_order=30)
    s.TextSprite("Нажмите ПРОБЕЛ для рестарта", pos=(s.WH_C[0], s.WH_C[1] + 50), font_size=20, color=(0, 0, 0), scene=self, sorting_order=30)
```

**Важно:** Строка `if self.is_game_over: return` — это защита! Без неё надписи создавались бы каждый кадр, пока птичка касается трубы.

# Слайд 10: Рестарт по пробелу
Если игра окончена, управление должно быть заблокировано. А если нажать ПРОБЕЛ — сцена перезапускается!

Ставим это **в самое начало** метода `update`:
```python
def update(self, dt):
    # --- GAME OVER: ждём рестарт ---
    if self.is_game_over:
        if s.input.was_pressed(s.pygame.K_SPACE) or s.input.was_mouse_pressed(1):
            s.restart_scene()
        return  # Код ниже НЕ выполняется!
```

`s.restart_scene()` — создаёт совершенно новую сцену с нуля. Все старые спрайты, трубы и надписи удаляются автоматически!
