# Слайд 3: Сохранение данных (save/load)
Когда мы закрываем игру, все переменные исчезают навсегда. Чтобы **сохранить лучший счёт**, запишем его в файл!

В SpritePro это делается двумя строками:
```python
# Сохранить:
s.save_load.save({"best": 10}, "best_score.json")

# Загрузить:
data = s.save_load.load("best_score.json", default_value={"best": 0})
```

`default_value` — что вернуть, если файла ещё нет (первый запуск игры).

# Слайд 4: Загружаем рекорд при старте
В `__init__` загружаем лучший счёт из файла:

```python
# 5. Лучший счёт (загружаем из файла!)
saved = s.save_load.load("best_score.json", default_value={"best": 0})
self.best_score = saved["best"]
```

Это вызывается **один раз** при создании сцены. Если файла нет — берём 0.

# Слайд 5: Красные частицы при смерти! 💥
Помните жёлтые искры при прыжке? Теперь создадим **второй эмиттер** — для смерти:

```python
# 7. Партиклы на смерть (красные)
self.death_emitter = s.ParticleEmitter(s.ParticleConfig(
    amount=25,                            # Больше частиц!
    size_range=(4, 8),                    # Крупнее
    speed_range=(100, 250),               # Быстрее
    angle_range=(0, 360),                 # Во все стороны!
    lifetime_range=(0.4, 1.0),            # Дольше живут
    fade_speed=300,
    colors=[(255, 80, 80), (255, 40, 40), (255, 160, 60)],  # Красно-оранжевые
))
```

В `trigger_game_over` добавляем: `self.death_emitter.emit(self.player.position)`.

У нас теперь **два эмиттера** — и каждый для своего момента! Можно создавать сколько угодно разных эффектов.

# Слайд 6: Tween-анимации (волшебные переходы) ✨
**Tween** (от «in-between» — промежуточный) — это плавное изменение свойства спрайта. Движок сам анимирует значение от текущего до нужного!

### Пульсация счёта при очке:
```python
# В блоке подсчёта очков:
self.score += 1
self.score_text.text = f"Счёт: {self.score}"
# Пульсация текста при новом очке!
s.tween_punch_scale(self.score_text, strength=0.3, duration=0.2)
```

`tween_punch_scale` — текст резко увеличивается и пружинит обратно. Это как награда — мозг «чувствует» успех!

### Затухание птички при смерти:
```python
# В trigger_game_over:
s.tween_alpha(self.player, to=80, duration=0.5)
```

`tween_alpha` — плавно меняет прозрачность (255 = видимый, 0 = невидимый). Мы делаем птичку полупрозрачной.

# Слайд 7: Экран нового рекорда! 🏆
При проигрыше проверяем: побил ли игрок свой рекорд?

```python
# Проверяем и сохраняем рекорд
is_new_record = self.score > self.best_score
if is_new_record:
    self.best_score = self.score
    s.save_load.save({"best": self.best_score}, "best_score.json")

# Показываем рекорд
record_color = (255, 215, 0) if is_new_record else (0, 0, 0)
record_prefix = "🏆 НОВЫЙ РЕКОРД! " if is_new_record else "Лучший: "
s.TextSprite(f"{record_prefix}{self.best_score}", pos=s.WH_C, font_size=24,
             color=record_color, scene=self, sorting_order=30)
```

**Золотой цвет** `(255, 215, 0)` и **эмоджи** 🏆 — маленькие детали, которые делают игру приятной!

# Слайд 8: Музыка и Звуки 🎵
Какая игра без звука? Добавим фоновую музыку и звуки действий:

```python
# Запуск фоновой музыки
s.audio_manager.play_music("Monkeys-Spinning-Monkeys(chosic.com).mp3")

# Загружаем звуки
self.snd_jump = s.load_sound("jump", "whoosh.wav")
self.snd_punch = s.load_sound("punch", "punch.mp3")

# Воспроизводим звук прыжка в update():
self.snd_jump.play()
```
Звуки мгновенно делают игру в 10 раз приятнее!

# Слайд 9: Что мы используем (Game Feel)
Посмотрите, сколько приёмов **профессионального геймдизайна** мы применили:

| Приём | Где используется | Зачем? |
|---|---|---|
| Тряска камеры | Столкновение | Ощущение удара |
| Звуковой эффект | Прыжок / Удар | Аудиальная отдача |
| Частицы (жёлтые) | Прыжок | Обратная связь действия |
| Частицы (красные) | Смерть | Драматический эффект |
| Пульсация текста | Новое очко | Ощущение награды |
| Затухание спрайта | Смерть птички | Визуальное завершение |

| Золотой текст | Новый рекорд | Мотивация играть ещё! |

Это **8 видов обратной связи**! Именно они отличают скучную программу от увлекательной игры.

# Слайд 12: Собираем .exe! 📦
Хотите отправить игру друзьям? Им не нужно устанавливать Python — соберём **готовый .exe**!

### 1. Устанавливаем PyInstaller:
```bash
pip install pyinstaller
```

### 2. Собираем:
```bash
pyinstaller --onefile --noconsole solution.py
```
- `--onefile` — всё в один файл
- `--noconsole` — без чёрного окна консоли

### 3. ⚠️ Важно! Копируем ресурсы!
После сборки ваш `.exe` появится в папке `dist/`. Но он **не найдёт картинки и звуки** без помощи!

Скопируйте все ресурсы (`background.png`, `bird.png`, `pipe.png`, а также ваши `.mp3` и `.wav` файлы) **в ту же папку**, где лежит `.exe`:
```
dist/
├── solution.exe
├── background.png
├── bird.png
├── pipe.png
├── Monkeys-Spinning-Monkeys(chosic.com).mp3
├── whoosh.wav
└── punch.mp3
```

Готово! Теперь можно отправлять папку `dist/` друзьям — игра запустится без Python! 🎮
