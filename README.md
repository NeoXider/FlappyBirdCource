# 🚀 Курс: Создание игры Flappy Bird на SpritePro 

![Мини-Хакатон](assets/course_banner.png)

Добро пожаловать в репозиторий 4-дневного мини-хакатона по созданию легендарного хита — **Flappy Bird**! 🎮

## 🏆 О курсе
Этот интенсивный курс предназначен для учеников 2-го года обучения Python (алгоритмика). За 4 урока мы пройдем путь от пустого окна до полноценной и динамичной игры, используя современный высокоуровневый 2D-движок **SpritePro**.

Главная особенность курса — **практика и динамика**. Вместо скучной теории мы сразу "оживляем" графику, работаем с гравитацией, физикой и вводом!

## 🎓 Структура хакатона:

- **Урок 1: База и Первые Прыжки** 🐣
  - Подключение движка (`pip install spritePro`).
  - Система сцен, координаты (`pos` — это центр!).
  - Z-index (сортировка отрисовки).
  - Физика (`s.physics`), границы окна и регистрация прыжка по клику или пробелу.

- **Урок 2: Трубы и Бесконечный мир** 🚧 *(В разработке)*
  - Генерация препятствий.
  - Движение окружения, чтобы создать иллюзию полета.
  - Удаление объектов за экраном (оптимизация).

- **Урок 3: Игровая логика и Очки** 💯 *(В разработке)*
  - Столкновения (Геймовер).
  - Подсчет очков (`Score`).

- **Урок 4: Полировка и Финал** 🌟 *(В разработке)*
  - Эффекты, анимации, перенос идей на новые уровни!
  
## 🛠️ Что нам понадобится
- Python 3+
- Движок [SpritePro](https://github.com/NeoXider/SpritePro) (`pip install spritePro`)

*Создано учителем алгоритмики специально для быстрых и крутых результатов!*
